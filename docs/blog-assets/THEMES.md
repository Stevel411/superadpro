# Blog Platform — Theme Reference Mockups

Canonical visual reference for the 5-theme launch set (locked 17 Jun 2026,
re-approved & built out 23 Jun 2026). These HTML files are the design source of
truth the rendering engine is built to reproduce. All render the same sample
blog ("The Maker's Field" / Alex Rivera) to prove the content-agnostic theme
system — the same posts flow through any theme, instant + lossless switch.

Every theme carries the non-removable "Powered by SuperAdPro" referral footer.
Member content is NOT brand-locked to cobalt/cyan — each theme has its own
tasteful palette + default font pairing (house lock applies only to dashboard
chrome + the footer mark).

| File | Theme | Style | Default font |
|------|-------|-------|--------------|
| theme-banner-home.html / -post.html | **Banner** (default) | Masthead + banner hero + grid feed; full post-reading view | Classic Serif (Merriweather) |
| theme-classic-sidebar.html | **Classic Sidebar** | Traditional: main column + sidebar widgets (About/Subscribe/Popular/Topics) | PT Serif |
| theme-journal.html | **Journal** | Minimal single column, distraction-free, literary | Spectral |
| theme-bento.html | **Bento** | Modern modular tile grid (featured 2x2 + tiles) | Space Grotesk |
| theme-cinematic.html | **Cinematic** | Dark premium, glowing gradient hero | Sora |

Deferred: **Glass** (frosted/mesh — needs cross-device hardening).
